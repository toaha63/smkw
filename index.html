About the Keyboard

Shahid Mughdho Keyboard is a Windows keyboard app, made by Hasin Israk Toaha (SS 2025), dedicated to all Martyrs' in the quota movement of Bangladesh 2024. I have added Bijoy Layout, Probhat Layout, and Gitanjali Layout for writing Bangla. You can write in Unicode. This software uses low-level keyboard hooks; for this reason, Windows Defender and some antivirus software might flag it as a virus. However, it is not a virus. You can use this software with safety. We don't collect user data!
